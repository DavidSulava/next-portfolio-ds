# 🚀 My Portfolio

My portfolio website built with **Next.js 16**, focusing on sleek UI and smooth user experience.

**🔗 [Live Demo](https://davidsulava.github.io/next-portfolio-ds/)**

## ✨ Features
*   **Next.js 16 & TS:** Turbopack by default, optimized performance and strict typing.
*   **React 19:** Latest React features and improved rendering.
*   **Framer Motion:** Fluid animations and micro-interactions.
*   **Multi-language (i18n):** English and Russian with next-intl v4.

## 🛠 Tech Stack
*   **Core:** React 19, Next.js 16, TypeScript
*   **Styles:** Tailwind CSS, PostCSS, Autoprefixer
*   **Motion:** Framer Motion 12
*   **i18n:** next-intl 4
*   **Linting:** ESLint 9 (flat config)
