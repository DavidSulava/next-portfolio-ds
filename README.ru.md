# 🚀 Современное портфолио разработчика

Мой сайт-портфолио, созданный на **Next.js 14**, с акцентом на стильный UI и плавный пользовательский опыт.

**🔗 [Live Demo](https://davidsulava.github.io/next-portfolio-ds/)**

## ✨ Возможности
*   **Next.js 14 & TS:** Оптимизированная производительность и строгая типизация.
*   **Framer Motion:** Плавные анимации и микро-взаимодействия.

## 🛠 Технологический стек
*   **Core:** React 18, Next.js 14, TypeScript
*   **Styles:** Tailwind CSS, PostCSS, Autoprefixer
*   **Motion:** Framer Motion
