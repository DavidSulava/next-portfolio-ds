# 🚀 Мой сайт Портфолио

Мой сайт-портфолио, созданный на **Next.js 16**, с акцентом на плавный UI.

**🔗 [Live Demo](https://davidsulava.github.io/next-portfolio-ds/)**

## ✨ Возможности
*   **Next.js 16 & TS:** Turbopack по умолчанию, оптимизированная производительность и строгая типизация.
*   **React 19:** Последние возможности React и улучшенный рендеринг.
*   **Framer Motion:** Плавные анимации.
*   **Мультиязычность (i18n):** Английский и русский с next-intl v4.

## 🛠 Технологический стек
*   **Core:** React 19, Next.js 16, TypeScript
*   **Styles:** Tailwind CSS, PostCSS, Autoprefixer
*   **Motion:** Framer Motion 12
*   **i18n:** next-intl 4
*   **Linting:** ESLint 9 (flat config)
